# Device Assets

Place device images here for the settings dashboard.

## Required Files

### MX Master 4
- `mx_master_4.png` - High resolution (580x580px minimum), transparent background
- Source: Logitech press kit or screenshot with background removed

### Installation Path
When installed, these files should be placed at:
- `/usr/share/juhradialmx/devices/mx_master_4.png`

### Image Requirements
- Format: PNG with transparency
- Size: 580x580px or larger
- Background: Transparent
- Style: Product shot, perspective view preferred
- Drop shadow: Added by CSS, don't include in image

### Fallback
If the device image is not found, the dashboard will display a placeholder text.
